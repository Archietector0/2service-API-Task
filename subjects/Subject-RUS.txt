Описание проекта:

Данный проект предполагает разработку двух основных сервисов: пользовательского сервиса и сервиса истории деятельности.
В задачи пользовательского сервиса входит обеспечение трех критически важных конечных точек:

Создание пользователя: Реализация функциональности для создания новых профилей пользователей.

Модификация пользователя: Разработка возможности модификации существующей информации о пользователе.

Поиск пользователей: Реализовать конечную точку для получения полного списка пользователей.

Кроме того, необходимо обеспечить бесперебойную связь между этими сервисами. В частности,
события создания пользователей и их последующие изменения должны аккуратно передаваться в сервис User Activity History.
Таким образом, обеспечивается надежная система отслеживания всех действий, связанных с пользователем.

Требования к сервису "История действий":

В частности, сервис "История действий" должен быть оснащен надежным механизмом для тщательной регистрации действий пользователя.
Доступ к этой истории должен осуществляться через конечные точки с возможностью фильтрации по идентификаторам пользователей.
Кроме того, она должна поддерживать постраничную навигацию, позволяя пользователям переходить от страницы к странице.

Рекомендации по реализации:
Связь между сервисами: Методы взаимодействия между сервисами являются гибкими и могут быть адаптированы в соответствии с
требованиями проекта. Строгих ограничений на протоколы связи не существует.

Выбор технологии:
Оба сервиса должны быть реализованы с использованием TypeScript (TS). Использование TypeScript повышает надежность и удобство сопровождения кода.
В качестве системы управления базами данных (СУБД) в проекте используется PostgreSQL. Такой выбор обеспечивает согласованность и надежность данных.
Соблюдение данных рекомендаций и требований позволит обеспечить бесперебойное обслуживание пользователей, дополненное надежным сервисом истории активности.
Для обеспечения долгосрочного успеха проекта при его реализации следует уделять первостепенное внимание четкости, масштабируемости и удобству обслуживания.
